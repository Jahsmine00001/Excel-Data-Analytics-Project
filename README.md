A three-part Excel project covering the full analytics workflow — from writing business functions, to building pivot tables, to delivering a fully interactive executive dashboard.

Each folder represents a distinct skill layer, building progressively toward a production-ready reporting solution.

## Project Overview

| Folder | Focus | Dataset | Key Skills |
|--------|-------|---------|------------|
| 01-Functions | Business questions answered with Excel formulas | HR Employee Data (266 records) | XLOOKUP, FILTER, SUMIFS, COUNTIFS, UNIQUE, Conditional Formatting |
| 02-Pivot-Tables | Data summarization and aggregation | Call Center Data | Pivot Tables, Slicers, Grouping, Cross-tabulation |
| 03-Dashboard | Interactive executive reporting dashboard | Call Center Data (1,000 records) | Dashboard Design, KPI Cards, Dynamic Charts, Slicers |

---

## Tools Used

- Microsoft Excel (Advanced)
- Pivot Tables & Pivot Charts
- Excel Functions: XLOOKUP, FILTER, SUMIFS, COUNTIFS, UNIQUE, INDEX+MATCH, IFERROR, SORT+TAKE, LARGE, MIN, MAX, MONTH
- Conditional Formatting
- Slicers & Interactive Controls
- Dashboard Layout & Data Visualization

---

## Key Outcomes

- Answered 10 progressive HR business questions using advanced Excel functions
- Built pivot-driven summaries across multiple dimensions (rep, city, gender, rating, time)
- Delivered a fully interactive Call Center dashboard tracking 1,000 records across KPIs, trends, rep performance and customer demographics
